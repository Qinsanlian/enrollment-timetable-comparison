# 手动回归测试清单

每次修改代码后，请执行以下测试用例，确保功能未被破坏。

## 基础环境
- [ ] `npm run dev` 启动成功，浏览器打开页面无白屏
- [ ] 控制台无红色报错，`__tsBridge` 对象可访问
- [ ] `__tsBridge.escapeHtml('<div>')` 返回 `&lt;div&gt;`

## 选课表
- [ ] 默认显示内嵌范本（20 门课程）
- [ ] 网课序号前显示 `*`
- [ ] 双击课程名称可编辑，按 Enter 保存，操作记录有更新
- [ ] 点击网课复选框，序号前的 `*` 同步显隐
- [ ] 点击非网课行，周课表对应格子闪动黄色高亮并滚动到可见区域
- [ ] 导入中文示例 Excel，数据正确替换，来源显示文件名
- [ ] 点击"恢复内嵌范本"，数据恢复默认 20 门课

## 周课表
- [ ] 点击"填入演示序号"，格子正确填充，显示课程名称、时间、地点、代号
- [ ] 上栏输入有效序号，显示课程信息；下栏自动显示（如有值）
- [ ] 上下栏各输入不同序号，格子分为上下两部分，中间虚线分隔
- [ ] 输入无效序号（如 99），格子变红并提示「序号应在 1～20 之间」
- [ ] 修正无效序号后，红色背景消失
- [ ] 点击 `+` 按钮添加第三栏，输入序号后三栏显示
- [ ] 点击 `-` 按钮移除第三栏，恢复双栏或单栏
- [ ] "按上课时间自动填格"功能正常，填充结果合理
- [ ] 溢出格（三门及以上）显示 `+N` 徽章，鼠标悬停显示未填入课程名
- [ ] 点选课表行，周课表对应格子闪动定位

## 撤销 / 重做
- [ ] `Ctrl+Z` 撤销周课表修改（输入或自动填格）
- [ ] `Ctrl+Y` 或 `Ctrl+Shift+Z` 重做
- [ ] 撤销/重做按钮在无历史时灰色，有历史时可点击

## 语言与界面
- [ ] 侧栏选择「English」，页面标题、按钮、表头等切换为英文，选课表数据不变
- [ ] 切回中文，一切恢复
- [ ] A3 预览区随窗口缩放，显示当前比例提示
- [ ] 修改课次时间参数（开始时间、每节课时长等），点击"计算并应用"，周课表节次时间更新
- [ ] 侧栏所有按钮功能正常（打印、导出、演示、保存、清空等）

## 导出
- [ ] 打印 / PDF：弹出检查清单 → 输入确认短语 → 打印预览为 A3，带水印
- [ ] 导出双语 PDF：生成两页 A3 PDF，中文 + 英文
- [ ] 导出申请包：生成封面（信息表） + A3 课表页

## 数据持久化
- [ ] 刷新页面后，周课表输入的数据仍然存在
- [ ] 导入 Excel 后刷新，选课表保持导入的数据
- [ ] "清除本机全部缓存数据"功能正常，清除后数据恢复默认
- [ ] 导出 JSON 备份 → 清空数据 → 导入备份，数据和课表完整复原

## 异常处理
- [ ] 页面翻译开启时，导出按钮被阻止或提示不可用
- [ ] 缺少 CDN 库时，相关按钮提示"未加载"
