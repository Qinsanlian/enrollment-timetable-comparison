# 课程 & 选课对照表 II

单文件项目：用浏览器直接打开 **`index.html`** 即可使用（无需构建、无需服务器）。

## 功能概要

- **选课表**：与教务导出的 **`选课表.xlsx`** 一致——无表头、固定 10 列（A 学年至 J 上课地点）；支持导入 / 本机保存；侧栏可下载同名示例文件。
- **周课表**：周一至周日共 7 列，单张 A3 竖版预览与打印。

## 文件说明

| 文件        | 说明           |
| ----------- | -------------- |
| `index.html` | 全部界面与逻辑 |

将本文件夹整体复制或打包分发即可。
